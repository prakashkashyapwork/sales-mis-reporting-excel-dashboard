# Retail Sales Analysis (Python + SQL)

## Project Overview
This project analyzes retail sales data to identify profit trends, loss-making products, and business insights.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SQLite)
- Power BI (for dashboard)

## Key Insights
- Central region has lowest profit performance
- High discounts reduce profitability
- Some products generate losses despite high sales
